# Interactive Particle Cloud Background

An elegant, lightweight, and high-performance interactive particle background built using Vanilla JavaScript and HTML5 Canvas. The particles dynamically follow the mouse cursor while maintaining an aesthetic distance, creating a fluid "particle cloud" or "galaxy trail" effect.

Inspired by the classic Google Antigravity visual style.

## ✨ Features

- **Dynamic Mouse Tracking:** Particles smoothly follow the cursor using linear interpolation (Lerp).
- **Aesthetic Spacing:** Trigonometric distribution (`Math.sin` & `Math.cos`) ensures particles maintain a beautiful cloud structure around the cursor instead of collapsing into a single point.
- **Zerolight Dependency:** Built entirely with native web technologies—no heavy libraries (like tsParticles or Particles.js) required.
- **Elegant Trail Effect:** Uses opacity blending on frame refreshes to leave smooth, ghost-like particle tails.
- **Responsive Design:** Automatically adapts and redraws upon window resizing.
